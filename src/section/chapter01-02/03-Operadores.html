Tipos de Operadores

Precisamos de operadores quando realizamos qualquer operação em uma linguagem de programação.

Operadores Aritmético

Operadores Descrição
+ Adição
- Subtração
* Multiplicação
/ Divisão
% Módulo( resto de uma operação de divisão)
++ Incremento
-- Decremento

Operadores de atribuição

Operadores Descrição
= Atribuição
+= Atribuição de Soma( x += y) == ( x = x + y)
-= Atribuição de Subtração( x -= y) == ( x = x - y)
*= Atribuição de Mutiplicação( x *= y) == ( x = x * y)
/= Atribuição de Divisão( x /= y) == ( x = x / y)
%= Atribuição de Resto( x %= y) == ( x = x % y)

Operadores de Comparação

Operadores Descrição
== Comparação (igual a)
=== Comparação que seja estritamente igual ao outro (Em valores e o tipo da variavel)
!= Negação (Diferente do outro)
> Maior que a outra variavel
>= Maior ou igual a outra váriavel
< Menor que a outra variavel
<= Menor ou igual a outra variavel

Operadores Lógicos

Operadores Descrição
&& (As duas condições precisam ser verdadeira) E
|| Ou (Uma das condições precisa ser verdadeira)
! Negação (Não pode ter o outro)

Operadores bit a bit

Operadores Descrição

Operadores bit a bit (bitwise)
