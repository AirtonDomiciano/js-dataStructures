Operadores

De acordo com a especificação, há dois tipos de dados em Js

  • Tipo de dados primitivos

    null (Nulo), undefined (Indefinido), string, number, booleano e symbol

  • Tipo de dados derivados/objetos

    objetos js, incluindo funções arrays e expressões regulares

Váriaveis

As váriavies armazenam dados que podem ser definidos, atualizado e recuperados sempre que necessário

Os valores atribuídos a uma variavel tem um tipo

  • Number (Número)
  • String (São texto)
  • Boolean (Apenas Aceita valores true ou false)
  • Function (Essa variavel é uma função)
  • Object (Objeto)

Verdadeiro e falso

Em js, true e false são um pouco complicados. Na maioria das linguagens, os valores booleanos true e false representam os resultados verdadeiro/falso. Em js uma string com Packt (Ou webpack é um empacotador de módulos estáticos para aplicações JavaScript modernas) é avaliada como true

Os valores atribuídos a uma variavel tem um tipo

  • Undefined (false)
  • Null (false)
  • Booleano (Verdadeiro ou falso)
  • Number (O resultado é falso para +0, -0 ou NAN; caso contrário, é true)
  • String O resultado é falso para string vazia
  • Object true
